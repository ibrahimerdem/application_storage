# EVALUATING CROSS-SELLING OPPORTUNITIES WITH RECURRENT NEURAL NETWORKS ON RETAIL MARKETING

## Abstract
---------------------------------------------------------------------------------------

<center>Recommender systems are considered to be capable of prediction
of what is the next product to buy for a specific customer. For an ef-
fective cross-selling framework it is crucial to identify which customers
are more suitable than others to targeting a product for retail indus-
try. This study proposes an extended version of an existing model
which implements a recurrent neural network architecture process-
ing an implicit feedback. It represents an alternative solution to a
well-known business problem: improving cross-selling effectiveness by
estimating customers’ likelihood for which products or services to buy
next time. A recommender system which employs a recurrent neu-
ral network with different data configurations is the core concept of
the framework. In this study we illustrate that the concatenation of
relevant sequential features may supply additional information to the
model, and increase the evaluation metrics.<center>
